Randomness Hunt

CS62 Applied Cryptography, Instructor: Sean Smith

The code does three major things. First, we use cryptographically strong rand_bytes() function to generate many samples. Then we use statistical methods to evaluate the given sequences. Finally, we determine whether each sequence is generated by human or machine. A detailed discussion can be found in "q4.txt".
